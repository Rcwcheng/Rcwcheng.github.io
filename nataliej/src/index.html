Natalie.J 是位對於AI, Computer graphics, Audio Visual 充滿喜愛與熱誠的藝術家和工程師。白天從事DeepLearning 結合Computer Generated Hologram 的研究，夜晚則專注於探索電腦圖學的世界，也進行AudioVisual的創作。

她曾經於友達光電擔任軟體工程師的實習（C#), 也曾在涅索開發(NAXS CORP) 從事Unity Shader 開發和前端工程。她會的程式語言有C, C#,Python, GLSL, JS,以及一些前端網頁語法。

她也曾參與燧人氏EBB.4 VJ演出，一開始是從Processing起家，近期則喜歡拿GLSL搭配Touchdesigner進行創作，她選擇GLSL的原因是由於她著迷於自然界中的美麗與神秘背後的數學與法則。

    Shader Tunnel 是一個實驗性的網站（持續在開發當中），它被Natalie.J用來研究有立體空間感的GLSL作品之演算法，以及探索藉由「網頁」之載體，與虛擬世界互動的體驗及可能性。

    網站是由Vue-cli 架構， 並以vue.js + three.js + WebGL撰寫而成，網站內目前除了有Natalie.J本人的創作，也有嘗試放上其他優秀的creator的作品做互動與體驗的實驗。
